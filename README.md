# Note Keeper

A new Flutter Note Keeper application made with SQLite. 

# Library used
1) sqflite
2) intl
3) path_proivder

# How to install on your system

1) Clone the repository on your system
2) Try flutter pub get command
3) Do flutter clean
4) flutter run
